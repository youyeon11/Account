<h3>기획자들의 요구 사항</h3>
> 우리 회사는 신규 사업으로 핀테크를 하려고 한다.
<p> > 핀테크를 하려면 제일 먼저 필요한 것은 바로 사용자들 마다의 잔액을 관리할 수 잇는 계좌 시스템! </p>
<p> > 계좌 시스템이 있어야 사용자의 잔액을 우리 회사에서 계쏙 관리하고 사용자들은 그 잔액을 쓰기 위해 우리의 핀테크를 자연스럽게 사용하게 된다.</p>
